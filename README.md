# Rick and Morty App

App para buscar personajes de Rick and Morty. He usado material ui para aprenderlo pro mi cuenta (solo he usado ia para eso)

## Que he usado

- **React** con TypeScript
- **Vite** con SWC
- **Material UI v5** para los componentes
- **Rick and Morty API**

## Cosas que he usado

### Material UI

- [Box](https://mui.com/material-ui/react-box/) - contenedor basico
- [Stack](https://mui.com/material-ui/react-stack/) - para layouts flexbox faciles
- [Grid](https://mui.com/material-ui/react-grid/) - para hacer el grid de las cards
- [Card](https://mui.com/material-ui/react-card/) - cards de los personajes
- [Chip](https://mui.com/material-ui/react-chip/) - para mostrar el status y especie
- [TextField](https://mui.com/material-ui/react-text-field/) - input de busqueda
- [Dialog](https://mui.com/material-ui/react-dialog/) - modal con el detalle
- [Button](https://mui.com/material-ui/react-button/) - boton de cargar mas
- [CircularProgress](https://mui.com/material-ui/react-progress/) - spinner de carga
- [Typography](https://mui.com/material-ui/react-typography/) - textos con estilos

### React

- Custom hooks (`useCharacters`, `useFocus`)
- Manejo de estado con useState
- Efectos con useEffect
- Refs con useRef

### La API

- [Rick and Morty API Docs](https://rickandmortyapi.com/documentation)
